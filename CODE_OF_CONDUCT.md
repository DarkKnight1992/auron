# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Pledge

We pledge to make participation in Auron a welcoming experience for everyone, regardless of background or identity.

## Standards

Expected behavior includes being respectful, constructive, and collaborative. Unacceptable behavior includes personal attacks, trolling, and unwelcome conduct.

## Enforcement

Reports of unacceptable behavior may be submitted to the project maintainers via GitHub issues marked confidential, or via email to the maintainer listed in the repository profile. All reports will be reviewed and investigated promptly and fairly.
