# Introduction to Auron

Auron is a Vue 3 component library delivering 85+ accessible, beautifully designed components powered by Reka UI and Tailwind CSS 4.

Built on [Reka UI](https://reka-ui.com) (the Vue equivalent of Radix UI) for accessibility primitives, [Tailwind CSS 4](https://tailwindcss.com) for styling, and [tailwind-variants](https://www.tailwind-variants.org) for type-safe variant definitions.

## Why Auron?

- **85 components**
- **Accessible by default** — every component passes `@chialab/vitest-axe` with zero violations
- **Vue 3.5+ native** — `useTemplateRef`, `defineModel`, `useId` — no Vue 2 compatibility baggage
